# stoksmart
## Deskripsi Proyek
StokSmart merupakan startup berbasis teknologi yang dirancang untuk membantu pemilik toko sembako dan UMKM dalam mengelola stok barang secara digital. Aplikasi ini bertujuan mengurangi kesalahan pencatatan stok yang masih dilakukan secara manual serta membantu pengguna memantau persediaan barang dengan lebih mudah dan efisien.
## Latar Belakang
Banyak pelaku UMKM dan pemilik toko sembako masih menggunakan buku catatan untuk mengelola stok barang. Cara tersebut sering menyebabkan kesalahan pencatatan, keterlambatan mengetahui stok habis, serta kesulitan dalam melakukan pengecekan persediaan barang.StokSmart hadir sebagai solusi digital yang sederhana dan mudah digunakan untuk membantu pengelolaan stok barang.
